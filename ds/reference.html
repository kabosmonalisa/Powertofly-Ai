Navigation
The nav is rendered per-page in HTML. ptf.css styles it; ptf.js handles hamburger + scroll transparency.
Default: transparent → frosted glass on scroll. Override with .nav { background: rgba(255,255,255,0.85); } for always-frosted (talents).

    <a class="brand" href="/">...logo... <span class="brand-ai">AI</span></a>
    <div class="nav-links">...links...</div>
    <div class="nav-cta">...buttons...</div>
