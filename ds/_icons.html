Icon gallery — MUI Outlined, treatment by size

One genuine MUI family: the Outlined variant. All the same outline style — no more mixing filled and outline. What changes is the treatment, by size: gradient only when big, plain ink when small.

THE SIZE RULE (2026-07-04): gradient only at ≥40px — big feature/hero icons & the thank-you check. Under 40px (list bullets, chips, inline labels, fields, buttons, stepper marks — anything repeated) icons are plain, single-color ink, no gradient. A gradient icon at 16–24px is mush.

4 · Small (<40px) — plain ink, NO gradient

Same MUI Outlined paths, but at list/chip/inline size they're plain single-color ink — the gradient is stripped. Colour via a token in CSS (fill: var(--ink)), never inline hex, never a gradient. This is how list bullets, chips, field icons, stepper marks and buttons render.
